# Pratyay360 Tap

## How do I install these formulae?

`brew install pratyay360/tap/<formula>`

Or `brew tap pratyay360/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "pratyay360/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
